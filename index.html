MY PROJECT

In this project, I created an android mobile application to complete my final college assignment and I use the Flutter programming language with the Firebase database to store and synchronize data in real-time.

This time the project is making a server for one of the foundation schools in Bogor, because at the moment all schools are recommended to use a computerized system in carrying out exams and at the same time for computer lab facilities. With a large number of funds, we collected all the tools and computer equipment needed to build a server, starting from PC, Laptops, LAN Cables, Hubs / Switches, Routers, Microtik, and modems.
